# Purpose

This document lists the official resources that AI agents should consult to understand each examination's:

- Exam structure
- Skills tested
- Scoring criteria
- Official sample questions
- Public preparation materials
- Official vocabulary lists (when available)
- Official grammar requirements
- Official assessment rubrics

These resources are for learning the exam formats only.

Never copy copyrighted questions, passages, audio, or answer keys.

Always generate original practice content inspired by the official exam specifications.

# AI Agent Rules

For every supported examination:

1. Read the official public documentation.
2. Learn the exam structure.
3. Learn the scoring rubrics.
4. Learn timing rules.
5. Learn section weights.
6. Learn CEFR mapping where applicable.
7. Generate ORIGINAL practice material.
8. Never reproduce copyrighted content.
9. Clearly distinguish between official information and AI-generated practice.
10. Keep resources up to date by periodically checking the official websites.

---

# Continuous Improvement

When a new exam version is released:

- Update section formats
- Update scoring rules
- Update timing
- Update sample task styles
- Revise AI prompt templates accordingly

Review official resources before each major application release.
