# AI Language Coach

Welcome to the **AI Language Coach** repository. This is an advanced, personalized language-learning platform designed to help users master spoken and written language skills. The platform bridges the gap between passive learning and active fluency, with a focus on preparation for major international examinations (IELTS, PTE, TOEFL, OET, Goethe, TestDaF, DELF, JLPT, and TOPIK).

## 🛠️ Technology Stack

*   **Frontend Client:** [Flutter](https://flutter.dev) (iOS / Android / Web)
*   **Backend & Infrastructure:** [Supabase](https://supabase.com) (Authentication, PostgreSQL Database, Realtime tables, Edge Functions)
*   **Voice Pipeline:** [LiveKit](https://livekit.io) + AI Realtime (Ultra-low latency STT and TTS streaming)
*   **Linguistic AI Core:** Gemini (Standard processing) & OpenAI (Premium evaluation and Mock grading)
*   **Telemetry & Analytics:** Firebase (FCM, Crashlytics, Analytics)
